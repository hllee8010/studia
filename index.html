Studia 이용약관 (초안)

⚠️ 본 약관은 초안(Draft)입니다. 실제 서비스 출시 전 반드시 변호사 검토 후 수정·게시해야 합니다. 현재 문서는 법적 효력을 보증하지 않습니다.

제1조 (목적)

본 약관은 Studia(이하 "서비스")가 제공하는 AI 기반 학습 보조 서비스의 이용 조건 및 절차, 이용자와 운영자의 권리·의무를 규정함을 목적으로 합니다.

제2조 (정의)

  • "서비스"란 Anthropic Claude API와 연동되어 사용자의 질문에 대한 학습 보조 응답을 생성하는 웹 애플리케이션을 말합니다.
  • "이용자"란 본 약관에 따라 서비스를 이용하는 개인을 말합니다.
  • "API 키"란 Anthropic사에서 이용자에게 발급하는 인증 문자열을 말합니다.

제3조 (서비스의 특성)

  • 본 서비스는 서버 저장소를 거치지 않고 이용자의 브라우저에서 Anthropic으로 직접 요청을 전송합니다.
  • API 사용 과금은 이용자가 Anthropic에 발급받은 키에 따라 이용자 본인에게 부과됩니다.
  • AI가 생성한 답변은 정확성을 보장하지 않으며, 학습 참고용으로만 사용되어야 합니다.

제4조 (이용자의 의무)

  • 이용자는 본인의 API 키를 안전하게 보관할 책임이 있습니다.
  • 이용자는 타인의 권리를 침해하거나 법령을 위반하는 목적으로 서비스를 이용하지 않습니다.
  • 공용 기기에서는 "이 기기에 키 기억하기" 옵션을 사용하지 않도록 유의해야 합니다.

제5조 (책임의 제한)

운영자는 AI 생성 결과의 정확성·완전성·적합성을 보증하지 않으며, 이로 인한 학업상·경제상 손해에 대해 책임지지 않습니다. 또한 이용자의 부주의한 API 키 노출로 발생한 과금에 대해 책임지지 않습니다.

제6조 (서비스 변경 및 종료)

운영자는 서비스 개선 또는 기술적 사유로 서비스 내용을 변경하거나 중단할 수 있으며, 주요 변경 사항은 사전에 고지합니다.

제7조 (준거법 및 관할)

본 약관은 대한민국 법령을 준거법으로 하며, 서비스 이용과 관련한 분쟁은 민사소송법상의 관할 법원을 제1심 관할로 합니다.

※ 본 초안은 작성일 기준 템플릿이며, 실제 시행 전 법률 전문가 검토가 필요합니다.

개인정보 처리방침 (초안)

⚠️ 본 방침은 초안(Draft)입니다. 개인정보 보호법·정보통신망법에 따라 법률 전문가의 검토가 반드시 필요하며, 실제 운영자의 상호·연락처·DPO 정보가 기재되어야 합니다.

1. 수집하는 개인정보 항목

본 서비스는 다음 항목을 이용자 기기(브라우저)에만 저장합니다. 서버로 전송·저장하지 않습니다.

  • Anthropic API 키 (선택: "이 기기에 키 기억하기"를 켠 경우에 한해 localStorage에 저장)
  • 이용자가 입력한 학습 질문·답변 내역 (오답노트, 세션 중 대화 기록)
  • 이용 약관·개인정보 동의 여부 및 동의 시각

2. 외부 전송 항목

  • 이용자가 입력한 질문·첨부 이미지는 AI 응답 생성을 위해 Anthropic(api.anthropic.com)으로 전송됩니다.
  • 외부 전송 시 Anthropic의 개인정보 처리방침이 함께 적용됩니다.

3. 개인정보의 보유 및 파기

  • 브라우저 localStorage에 저장된 정보는 이용자가 "동의 철회" 또는 "키 변경"을 선택하거나 브라우저 데이터를 삭제할 때 즉시 파기됩니다.
  • 운영자는 이용자의 학습 데이터를 서버에 별도 저장하지 않습니다.

4. 만 14세 미만 아동에 관한 사항

본 서비스는 만 14세 이상을 대상으로 합니다. 만 14세 미만 이용자는 「개인정보 보호법」 제22조에 따라 법정대리인의 동의가 필요하며, 본 동의 화면에서 연령 확인을 실시합니다.

5. 이용자의 권리

이용자는 언제든지 자신의 개인정보에 대한 열람·정정·삭제·처리정지를 요구할 수 있습니다. 본 서비스에서는 "동의 철회" 버튼 및 "오답노트 비우기" 기능을 통해 이를 즉시 실행할 수 있습니다.

6. 쿠키 및 분석 도구

본 서비스는 내부 이벤트 트래킹(window.Studia.track)을 수행하지만, 외부 분석 도구로 전송하지는 않습니다. 향후 상용화 시 Google Analytics 등 외부 도구 연동 계획이 있을 경우 별도 고지 및 동의를 받습니다.
